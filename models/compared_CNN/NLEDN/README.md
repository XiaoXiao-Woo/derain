### Running Steps
1. Download four testing sets (Rain100L, Rain100H, DDN, DID-MDN).
2. Choose the type of dataset in **config.py**.
    ```
    data_type_list = ['Rain100L', 'Rain100H', 'DDN', 'DID']
    data_type = data_type_list[2]
    ```
3. Run **test.py**.
    ```
    python test.py
    ```
